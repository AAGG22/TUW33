Estructura del Control while en Java

El condicional if en Java se utiliza para ejecutar un bloque de código si se cumple una condición específica. Aquí tienes un ejemplo:


public class Main {
    public static void main(String[] args) {
        int num = 10;
        
        if (num > 0) {
            System.out.println("El número es positivo.");
        } else if (num < 0) {
            System.out.println("El número es negativo.");
        } else {
            System.out.println("El número es cero.");
        }
    }
}
